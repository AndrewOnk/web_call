# README

This is a Ruby on Rails + ReactJs application that implements WebRTC via STUN 

* Ruby version: 2.5.7

* System dependencies:
- node 14.1.0
- npm 6.14.4
- ruby 2.5.7
- Ruby on Rails 6.0.3
- ReactJS 16.13.1
- webpack 4.43.0

* Configuration
- gem install bundler
- git clone
- cd web_call
- bundle
- yarn install
- rails s
